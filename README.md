# AI-SALES-DATA-ANALYSIS-USING-CHATGPT
This project analyzes retail sales data to identify which product category the business should invest in next, based on real sales performance, customer demographics, demand trends, and price sensitivity. It includes data processing, exploratory analysis, visualizations, dashboards, and a final investment recommendation.
